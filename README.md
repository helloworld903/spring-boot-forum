# spring boot论坛(spring boot+mysql+thymeleaf)
本项目为本人用于熟悉spring boot框架设计模式以及web开发的练手项目，也比较适合新手入门，涉及的技术有

 1. spring boot框架
 2. mysql+redis
 3. thymeleaf引擎
 4. JavaMailSender
 5. JPA
 6. ORM
 7. lombok
 
目前，该论坛已经实现的功能有
 - [ ] 用户注册、登录、激活、找回、信息修改
 - [ ] 发表说说、展示、修改
 - [ ] 评论
 - [ ] 点赞
 - [ ] 消息通知
